# Icônes — Gestio V4

> Génère les icônes pour Desktop et Mobile.

## Générer les icônes

```bash
uv run python resources/icons/generate_icons.py
```

| Fichier | Usage | Généré par |
|---------|-------|------------|
| `gestio.png` | Linux, Android | `generate_icons.py` |
| `gestio.ico` | Windows | `generate_icons.py` |
| `gestio.icns` | macOS, iOS | `generate_icons.py` |

## Pour Mobile

Les icônes sont aussi utilisées par Capacitor pour iOS/Android.
